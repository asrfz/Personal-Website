---

I made this site using React and Vite. Run it locally, then ship the `dist` folder from `npm run build`.

## Setup

```bash
npm install
```

## Commands

| Command        | What it does              |
| -------------- | ------------------------- |
| `npm run dev`  | Dev server (hot reload)   |
| `npm run build`| Production build → `dist/`|
| `npm run preview` | Serves `dist` locally  |
